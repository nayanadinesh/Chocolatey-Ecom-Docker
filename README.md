# 🛒 MERN Stack Web Application

## 🔍 Overview

This project showcases scalable architecture, RESTful API design, and seamless client-server interaction.

## ⚡ Installation

### Clone the repository:
```bash
git clone <your-repo-link>
cd <repo-name>
```

---

### Install dependencies:

#### Backend:
```bash
cd server
npm install
```

#### Frontend:
```bash
cd client
npm install
```

---

## ▶️ Usage

### Start backend server:
```bash
cd server
npm start
```

### Start frontend:
```bash
cd client
npm start
```

App runs on:
```
http://localhost:3000
```
